# Visione

RetroNet nasce per ricostruire l'evoluzione dell'informatica in modo pratico, leggibile e sperimentabile.

La traiettoria del progetto parte dal basso livello:

```text
Intel 4004 -> Intel 8008 -> Intel 8080 -> CP/M-like
```

e arriva ai sistemi collegati in rete:

```text
terminali -> BBS -> Gopher -> HTTP storico -> primo Web
```

## Obiettivo

L'obiettivo e costruire un laboratorio educativo in cui ogni pezzo sia:

- piccolo abbastanza da essere capito;
- testato abbastanza da essere affidabile;
- documentato abbastanza da essere usato da altri;
- integrabile in una demo completa.

RetroNet deve funzionare sia come portfolio tecnico sia come progetto open source contributor-friendly.

## Esperienza desiderata

La prima esperienza pubblica consigliata e **RetroNet 4004 Web Lab**:

```text
RETRO-NET 4004 LAB
CPU: Intel 4004
Status: running
ROM: examples/demo.rom

PC=000 ACC=0 CY=0
PC=001 ACC=5 CY=0
PC=002 ACC=9 CY=0
DONE
```

Un utente deve poter avviare il laboratorio, aprire una dashboard e osservare una CPU storica eseguire un programma in un terminale web.

## Regola fondamentale

Meglio un modulo piccolo ma completo che dieci moduli iniziati e lasciati a meta.
